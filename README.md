# 🥖 MBS Patisserie — Page vitrine HTML & CSS

Ce mini-projet a été réalisé dans le cadre de la formation. L’objectif était simple mais formateur : créer une page vitrine en combinant HTML et CSS. L'idée ? Une boulangerie-pâtisserie fictive baptisée **"Au MBS"**.

## 🎯 Objectifs pédagogiques

- Découverte des bases de la structure HTML
- Utilisation du CSS pour styliser les éléments
- Intégration d'images, mise en page, création de sections

## ✨ Ce que propose la page

- 🏠 **En-tête visuelle** avec la devanture de la boulangerie
- 🔗 **Barre de navigation**  entre Accueil, Boutique et Contact
- 🧁 **Galerie de produits** gourmands : tartelettes, macarons, croissants...
- 📬 **Formulaire de contact** 

## 🛠️ Technologies utilisées

- HTML
- CSS
